# Overview
This project contains a fully functional Active Directory lab built to simulate common enterprise attack paths, and defensive strategies. The lab includes a Windows Domain Controller, a Windows client, and a attacker machine running Kali Linux.

# Environment
- Server running VMware ESXi 8.
- Windows Server 2022 Standard.
- Windows 10 Pro
- Kali Linux.
- Domain: testad.net
- Network: Machines are isolated in their own vSwitch; not connected to other devices.
